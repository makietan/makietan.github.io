正直 GitLab は Github に名前が似ているだけです

2018-06-08 00:00

tech

GitLab とは

GitHub→GitLabの流れでGitLabがオープンソースプロジェクト向け無料キャンペーン - ITmedia NEWS

GitLabは2014年創業のサンフランシスコに拠点を置く非公開企業。GitHubと同様の、コードリポジトリのホスティングサービスを提供する。（ロゴの動物はキツネではなくタヌキ。）

だってさ。

Github と GitLab どちらが便利？

今まで Github と BitBucket を利用していた人ならわかると思うが、後者寄りです。 なので、今すぐに頑張って移行する必要はないと思います。

個人的には　Github が Microsoft に買収されて思うこと | 日報 なんて記事を書いたが、それでも GitLab に移行しようなんて1ミリも思わなかった。まだまだ、Github のほうが便利なので。 データの管理をするサービスなのにデータを紛失するようなところは個人的には候補にならない。
