## MAIN【高機動Ⅲ】
### 気力120以上のとき
- 運動性が20%増加する。

```
(X + 291) × 20% = 0.2X + 58.2
```

このパーツから得られる運動性は**349.2**となる。

## SUB【運動性プラスⅢ】
- 運動性 +25

```
291 + 25 = 316
```

このパーツから得られる運動性は**316**となる。
