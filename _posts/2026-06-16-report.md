Copilot の料金どうなってるか確認したときに、Code Review model という項目があって、これが何なのか気になったので調べてみた。

Code Review model とは、GitHub Copilot の機能の一つで、コードレビューを支援するための AI モデル。
Pull Request を作成した際に、コードの変更点を分析して、コードレビューのコメントを自動生成してくれるアレのこと。

だいたい1回のコードレビューで70円程度のコストがかかるっぽい。
まぁ、そこそこ高いけど、コードレビューの工数を削減できるなら十分元は取れるかなという感じ。
