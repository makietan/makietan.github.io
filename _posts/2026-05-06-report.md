まず、二郎系の呪文を関数化してみる。

返り値はオブジェクト型になっていないといけない。

C++あたりで書くと、以下のような感じになる。

```cpp
struct JiroOrder {
    std::string yasai;
    std::string ninniku;
    std::string abura;
    std::string karame;
};

JiroOrder check_ninniku_entry(std::string yasai, std::string ninniku, std::string abura, std::string karame) {
    JiroOrder order;
    order.yasai = yasai;
    order.ninniku = ninniku;
    order.abura = abura;
    order.karame = karame;
    return order;
}
```

いや、複雑だなおい。
