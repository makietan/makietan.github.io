## スクリプト

ファイル名を表示する

```
git status --porcelain | sed s/^...// | xargs -n 1 sh -c 'echo $0'
```
## 解説

変更があったファイル一覧はこれで表示可能

```
git status --porcelain | sed s/^...//
```

それを引数にして何かを実行する部分が以下

```
... | xargs -n 1 sh -c 'echo $0'
```
