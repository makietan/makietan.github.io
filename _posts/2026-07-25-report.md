## ラーメンを食べに行きたい

田町にもラーメン屋はあると思うので安価に移動して食べに行きたい。

ちぃばすに芝ルートという田町と新橋を結ぶルートがあるのでこれを使って行ってみる。

ちぃばすは1回100円で乗れるので安価に移動できる。

往復で200円で行ける。

電車だと山手線で160円（5分）、都営三田線で180円（5分+徒歩3分）、都営浅草線[三田駅下車]で180円(4分+徒歩3分)。



## 最終便

とりあえず最終便でケツを決める。

```
20:15 田町駅東口
| 芝ルート [みなとパーク芝浦->新橋駅] 新橋駅行
20:46 西新橋二丁目
```

その前の便は 19:50(31分)、19:30(39分)、19:05(33分)。

早く食べたら 19:50 の便で帰れる。

## ラーメンを30分（最大1時間）で食べる想定

ラーメンなので回転率は高いと思うので30分で食べる想定で行く。

20:15から30分引くと19:45に田町駅東口に着く必要がある。

2案ある。

```
18:36 西新橋二丁目
| 芝ルート [新橋駅->みなとパーク芝浦] みなとパーク芝浦行
19:15 田町駅東口
```

```
18:56 西新橋二丁目
| 芝ルート [新橋駅->みなとパーク芝浦] みなとパーク芝浦行
19:35 田町駅東口
```

バスなので遅延を考えると、18:36の便で行くのが無難だと思う。

18:11(39分)の便もある。

帰りの最終便は逃すが、

19:21(39分)、19:46(39分)、20:01(39分)、20:31(39分)で行くこともできるので、行きはちぃばすで行って、帰りは電車で帰るのもありだと思う。

## まとめ

仕事もあるので18:56で行ってみようかなと思う。

東口には、

- [ ] 田町えっちゃんラーメン。
- [ ] 玉 田町店

西口には、

- [ ] 三田製麺所 三田本店
- [✓] むらさき山
- [ ] 大輝家直系 家系ラーメン 大総家
- [ ] 中華そば専門 田中そば店 田町タワー店
- [ ] 舎鈴 田町駅前店
- [ ] らぁ麺 六花-rikka-

新橋周辺に昔いたお店がちらほらいたりするので久々に食べてみたい。

- [✓] ラーメン二郎 三田本店

すこし距離はあるが、 ラーメン二郎 三田本店 もまた行ってみたい。

行く気は一切ないけど、 日高屋 田町（西・東）口店 もある。

他にも色々あるから、ラーメン以外も食べてみたい。

麻布ラーメン 慶応三田店 が閉店して久しい...。
