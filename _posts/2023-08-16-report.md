## MAIN【アタッカーⅡ】
### 気力が120以上のとき
- 攻撃力が15%増加する。
#### 注釈
このパーツの攻撃力は2460なので実際には以下の通りである。

```
(X + 2460) * 15% = 0.15X + 369
```

このパーツから得られる攻撃力は**2829**となる。

## SUB【攻撃力プラスⅡ】
- 攻撃力 +175

```
2460 + 175 = 2635
```

このパーツから得られる攻撃力は**2635**となる。

## 総評

ステータスが低く採用するメリットが一切ない。
