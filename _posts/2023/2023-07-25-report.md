## MAIN【アタッカー（小）】
### 気力が120以上のとき
- 攻撃力が5.5%増加する。
#### 注釈
このパーツの攻撃力は1140なので実際には以下の通りである。

```
(X + 1140) * 5.5% = 0.055X + 62.7
```

このパーツから得られる攻撃力は**1202.7**となる。

## SUB【攻撃力プラス（小）】
- 攻撃力 +75

```
1140 + 75 = 1215
```

このパーツから得られる攻撃力は**1215**となる。

## 総評

ステータスが低く採用するメリットが一切ない。
メインにつけるメリットが一切ないユニットパーツ。
