## MAIN【狙い撃ち（小）】
### 気力が120以上のとき
- 照準値が5.5%増加する。
#### 注釈
このパーツの照準値が99なので実際には以下の通りである。

```
(X + 99) * 5.5% = 0.055X + 5.445
```

このパーツから得られる照準値は**104.445**となる。

## SUB【照準値プラス（小）】
- 照準値 +10

```
99 + 10 = 109
```

このパーツから得られる照準値は**109**となる。

## 総評

ステータスが低く採用するメリットが一切ない。
メインにつけるメリットが一切ないユニットパーツ。
