```json
{
  "filters": {
    "whitelist": {
      "allow": [
        "日本学生支援機構",
        "上限固定変動性"
      ]
    }
  },
  "rules": {
    "preset-ja-spacing": true,
    "preset-ja-technical-writing": {
      "no-exclamation-question-mark": false
    },
    "spellcheck-tech-word": true
  }
}
```

各種ライブラリに対して設定する必要がある。

```json
{
  "rules": {
    "no-exclamation-question-mark": false
  }
}
```

このようにtextlintに直接適用してもエラーは解消されなかった。
